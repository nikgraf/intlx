Dart Documentation › relative_time_locale_data

relative_time_locale_data library

Exposes LocaleData constants for use with DurationFormat and/or AgeFormat. For example:

import 'package:intlx/relative_time_locale_data.dart' as relative_time_data;
main() {
  relative_time_data.EN.load();
  relative_time_data.DE.load();
  // do something with [DurationFormat] and/or [AgeFormat].
}
