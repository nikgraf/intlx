iterable_locale_data library

Exposes LocaleData constants for use with IterableFormat. For example:

import 'package:intlx/iterable_locale_data.dart' as iterable_data;
main() {
  iterable_data.EN.load();
  iterable_data.DE.load();
  // do something with [IterableFormat].
}
